# Roady Car 🚗💨

**Roady Car** is an innovative smart car project developed by the **ELECTRODECLIC CLUB**. It is designed to evaluate road quality in real-time and provide a performance grade based on the terrain it traverses.

## 🌟 Features

- **Road Analysis**: Evaluates and marks the road quality.
- **Real-time Visualization**: Draws performance and data curves.
- **Audio Feedback**: Plays "funny sounds" based on its state or environment.
- **BLE Controlled**: Fully driven via Bluetooth Low Energy (BLE).
- **Smart Connectivity**: Interactive mobile interface for control and data monitoring.

## 🛠️ Technology Stack

- **Framework**: [Flutter](https://flutter.dev) for the cross-platform mobile application.
- **Communication**: Bluetooth Low Energy (BLE).
- **Hardware**: Built for electronic smart car kits.

## 🚀 Getting Started

To get started with the Roady Car mobile application:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-repo/roady_car.git
    ```
2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
3.  **Run the app**:
    ```bash
    flutter run
    ```
